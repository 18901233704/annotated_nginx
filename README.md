# annotated_nginx
Annotated Nginx Source（中文）

# 简介
注解Nginx源码，帮助学习Nginx

当前使用的是1.8.1稳定版，待注解较完善后升级至1.9.x。

Nginx1.9.11的变动较大，增加了动态模块，完善了多线程，值得仔细研究。

请参考《Nginx模块开发指南：使用C++11和Boost程序库》。
